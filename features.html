1#

creat rooms

with RoomBot, your server's user base can easily
create public and private rooms.
with custom rooms, your user base can form
communities and get to know each other with
more intimacy the room creation process is easy and fast.

2#

set limit to room members

once you created a room, public or private, to keep the intimacy and
the connection between the group members,
you can set the member limit as
you please. if you don't want a member limit,
you can type "na" at the setting process of the room

3#

change the room's role color

to give a feeling of personality to the room,
it's possible to change the room's role color to a custom one.
only the room's owner (the user who created the room)
can change the room.

4#

leaving / deleting a room

all of the room members are able to leave the room as they please.
only the room's owner can delete the room.
when the room's owner leaves the room, the user who
joined the first becomes the room's owner.
if there is only one user in the room and he leaves,
the room will delete automatically.

5#

rooms moderation

to keep all of the rooms safe and friendly place,
all of the users who have the admin role in the
server can see and monitor all of the rooms, and delete any rooms
they want even if they are not the room's owner!
